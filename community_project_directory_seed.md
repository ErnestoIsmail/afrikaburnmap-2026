# AfrikaBurn Community Project Directory Seed

Seed notes gathered from the AfrikaBurn Community Project Directory and related project pages.

Scope:
- camp and project level information only
- crew-signup and recruitment text intentionally omitted
- current-map matches added when the project name appears in `data.js`
- this is a first-pass research note; the complete structured directory export used by the app is in `project_details.js`

Source pages used:
- https://www.afrikaburn.org/event/participation/community-project-directory/
- https://www.afrikaburn.org/project-directory/
- individual project pages under `https://www.afrikaburn.org/project_directory/`
